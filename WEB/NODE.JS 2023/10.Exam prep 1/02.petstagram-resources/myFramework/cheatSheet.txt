2. Install & setup express

    * add routes
    - add static(from exam resources) to "public folder"
    - create "views" folder and put exam html's in it!
3. Add view engine: npm i express-handlebars
    * register with express
    * add views folder
    * add main layout   /src/views/layouts/main
    * render home template 
    * add partial template folder
4. Add home controller
    * add controller to routes
5. Connect database
    - install mongoose
    - connect database
    * set strict query / deprecation warning
6. Authentication
    * fix html links in layout

    * add register page
    * add login page

9. Install bcrypt and cookie-parser and configure

12. Generate jwt token
